VisIt Executables

This page contains the current version of VisIt that you can download for Unix, Windows, and Mac OS X systems. Older versions can be downloaded directly from SourceForge.net. For VisIt documentation, see the VisIt User Manuals. To check the integrity of the executables, see the visit_md5_checksums, the visit_sha1_checksums, or the visit_filesizes.

  • Unix—To install VisIt on a Unix system (including Linux, Solaris 2.7, AIX 5.1, Irix 6.5, Tru64 V5.1, and Mac OS X), you will need to download the appropriate distribution file and the visit-install script. Instructions for installing VisIt on a Unix system can be found in the Install Notes.

  • Windows—To install VisIt on a Windows system, you will need to download the Windows distribution file. Instructions for installing VisIt on a Windows system are also found in the Install Notes.
