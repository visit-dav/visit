VisIt 1.11.1 Release Notes

Welcome to VisIt's release notes page. Release 1.11.1 is primarily a bug fix release to release 1.11. Look here to see a complete list of bugs fixed in release 1.11.

Bugs fixed in version 1.11.1

  • A bug was fixed where scalable rendering (the parallel rendering mode) would produce the wrong image under a variety of conditions, including zooming in too far or clipping away part of the data set.
  • A bug was fixed where a slice that lay between two, flat domain boundaries could lead to a double contribution (one from each boundary). The images from VisIt were basically unaffected, but integrations (Weighted Variable Sums) and 2D area queries were double counting.
  • The host profiles for LLNL's juno system have been corrected to handle node names with 4 digits.
  • The source code was modified so that VisIt would compile with the gcc 4.3.x compilers.
  • The help files have been made available once again for the windows platform.
