VisIt 1.11.2 Release Notes

Welcome to VisIt's release notes page. Release 1.11.2 is primarily a bug fix release to release 1.11. It fixes additional bugs in release 1.11. Look here to see a complete list of bugs fixed in release 1.11.

Enhancements in version 1.11.2

  • Added support for ANNOTATION_INT nodelists in Silo plugin.

    • Bugs fixed in version 1.11.2

    • The file depcomp was added to the binary distributions so that plugins could be built against the binary distributions.
  • Fixed speed issues seen on a Silo file, using the HDF driver, with lots of domains and the standard (non-packed, directory-based) structured connectivity info
  • Fixed problem with revolved surface area query returning 0 on a mesh of beam elements. Note, it is important for the database to indicate that topological dimension of the mesh is 1. For a Silo database, one would use the DBOPT_TOPO_DIM option to the DBPutUcdmesh() call.
