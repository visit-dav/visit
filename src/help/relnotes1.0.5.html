VisIt 1.0.5 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.0.5

  • VisIt now displays a progress window for launching compute engines and metadata servers. The window is displayed if launching a process takes longer than 6 seconds and it features a cancel button that allows the user to cancel the process launch and return to VisIt.
  • Mesh plots are faster since normals are only calculated when needed.
  • 2D plots are generated faster.
  • The Main Window's Windows menu now has an option to clone a window which creates a new window containing the plots and settings of the active window.
  • Plots can now be copied to other windows via a new option in the Windows copy menu.
  • Windows that are posted on startup are now the right size.
  • A better error message is displayed for negative log Pseudocolor plots.
  • Reference lines now have an option to display labels.
  • VisIt now supports Hydra style interdomain connectivity.

Bugs fixed in version 1.0.5

  • VisIt sometimes has trouble launching its metadata server and compute engine on remote computers.
  • The view sometimes resets unexpectedly.
  • Unexpected error message with Pseudocolor plot.
  • Caching material interface reconstruction in parallel when generating ghost zones is broken.
  • The compute engine sometimes crashes when creating ghost zones.
  • The compute engine sometimes crashes when releasing data.
  • Large memory footprint for material interface recontruction.
  • Index Select of a material Subset plot does not work.
  • Lineout is incorrect.
  • Vectors are not properly drawn for some 2D curvilinear databases.
  • Odd failure when animating Pseudocolor plot.
  • Lighting affects 3D text.
  • Saving out multi-part files creates .visit files that cannot be read.
  • The Windows version now has a single Windows taskbar entry.
  • Transparency can get screwed up about variable centering.
  • Limits are grayed out in the Contour plot window when they do not apply.
  • The active tab in the view window change each time the user changes a field.
  • Crystal Eyes stereo mode now works.
  • Locking windows skips some windows after windows have been deleted.
  • Moving colortable endpoints now has an effect.
  • The Rendering window's stereo buttons are disabled when stereo rendering is turned off.
