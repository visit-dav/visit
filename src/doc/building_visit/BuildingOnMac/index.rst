Building on macOS with masonry
==============================

In this chapter, we will discuss how to build VisIt_ on macOS using **masonry**.
Masonry is a collection of Python scripts and JSON files that use ``build_visit``, and other 
system tools, to create a macOS Disk Image File (DMG).
