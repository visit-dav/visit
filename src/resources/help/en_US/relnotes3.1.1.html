VisIt 3.1.1 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Sections

  • Bug Fixes
  • Enhancements
  • Changes for VisIt developers

Bugs fixed in version 3.1.1

  • Corrected a bug where certain types of mili datasets would cause unexpected crashes.
  • Corrected a bug where the Data level comparison wizard would always enter zero for the simulation cycle.
  • Corrected a bug where zone pick highlights were incorrect after transforms were applied.
  • Summing mesh_quality/volume in a sub-material setting will now return the correct volume.
  • Corrected a bug where VisIt crashes on start-up on some Windows systems when the system OpenGL version is too old. Added a test of the OpenGL version during install, and use Mesa3D as a drop-in replacement if needed, with a warning on the installer pages to notify user that graphics cards / drivers should be updated.
  • Importing remote profiles from a Windows system was fixed.
  • Corrected a bug performing QueryOverTime with plots involving operator-created expressions.
  • Corrected a bug where VisIt would only display a black window when displaying on a Windows system running the XWin-32 X Server.

Enhancements in version 3.1.1

  • Updated the mfem library to version 4.0.
  • Added host profiles for Leibniz Supercomputing Centre."
  • FFP database plugin was updated and enhanced to use libstripack when it is available (either installed to VisIt's lib dir or path specified via VISIT_FFP_STRIPACK_PATH environment variable).

Changes for VisIt developers in version 3.1.1

  • Updated build_visit to work on macOS 10.14
  • Modified build_visit to download the visit source code and third party libraries from GitHub instead of NERSC."
